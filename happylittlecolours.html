Happy Little Colours

Visualisation

This project is a data visualisation study inspired by Bob Ross' distinctive colour palette. Using palette data gathered from all 31 seasons of Joy of Painting, the work reconstructs each season's set of hues as a Voronoi diagram, blending artistic interpretation with algorithmic structure. Each cell in the diagram represents a specific paint colour Ross used during that season, allowing viewers to observe patterns, stylistic evolution, and recurring tones across his decades of work.

Note: This visualisation tracks the number of paintings per season where each colour appeared, not the proportion of paint within each artwork.
