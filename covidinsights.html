COVID-19 Insights

Dashboard Application

This project is an interactive COVID-19 analytics dashboard application created to help users explore the pandemic through clear, data-driven visualisations. Designed during a period of rapidly changing information, the app consolidates global datasets into a unified interface that allows users to compare countries, track trends, and understand the broader context of COVID-19 spread and detection.

This dashboard, created as part of my Higher National Diploma coursework, was selected to be showcased in the national exhibition Sri Lanka Skills Expo 2023.
